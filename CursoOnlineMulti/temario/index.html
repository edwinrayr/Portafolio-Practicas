Plan de Estudios 2026

De Desarrollador a Arquitecto
Módulo por Módulo

Una ruta estructurada para dominar la arquitectura de software sin perderte en la teoría.

01

Fundamentos de Arquitectura de Software

Semana 1 - La Base Sólida
  • Diferencia entre Diseño de Código y Arquitectura de Sistemas.
  • Atributos de Calidad (Disponibilidad, Escalabilidad, Mantenibilidad).
  • Clean Architecture: Teoría y Práctica en proyectos reales.
  • Principios SOLID aplicados a nivel de componentes.
02

Patrones de Arquitectura Distribuida

Semana 2 - Escalando la Aplicación
  • De Monolito a Microservicios: Estrategias de descomposición.
  • Comunicación Síncrona (REST/gRPC) vs Asíncrona (Event-Driven).
  • Patrón Saga y CQRS para transacciones complejas.
  • Teorema CAP y consistencia eventual en Bases de Datos.
03

Infraestructura y Seguridad

Semana 3 - Cloud Native & DevSecOps
  • Contenerización profesional con Docker.
  • Orquestación básica con Kubernetes para desarrolladores.
  • Seguridad Zero Trust: Implementación de OAuth2 y OpenID Connect.
  • Pipelines de CI/CD: Del commit a producción.
04

Proyecto Final Integrador

Semana 4 - Manos al Código

Diseñaremos y programaremos el MVP de una plataforma de Streaming escalable (tipo Netflix Clone) aplicando todo lo aprendido.

  • Definición de requisitos no funcionales.
  • Diagramas C4 del sistema.
  • Revisión de código en vivo y feedback.
  • Certificación de Arquitecto Junior.
